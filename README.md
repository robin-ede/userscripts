# userscripts

Browser userscripts for converting rental prices to USD.

| Script | Site |
|--------|------|
| [realestate-nzd-to-usd](realestate-nzd-to-usd.user.js) | realestate.co.nz |
| [realestate-aud-to-usd](realestate-aud-to-usd.user.js) | realestate.com.au |

## Install

Requires [Tampermonkey](https://www.tampermonkey.net). Click a script file above, then **Raw**, then install.
